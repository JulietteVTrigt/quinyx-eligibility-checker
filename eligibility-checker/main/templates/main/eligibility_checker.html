Eligibility Checker

This tool will check if a user is eligible to work a shift. Note that this tool does not consider the rule logic. So if a shift is not assigned to a user because of a rule, this tool will not be able to tell you why. There are numerous reasons why a shift may not be assigned to a user:

1. Eligible Agreement
2. Active Department
3. Shifts within Availability
4. Shift outside Unavailability
5. Eligible by contract coverage
6. Eligible by shift department
7. Eligible by shift skills
8. Eligible by shift licences

You can check this Confluence page for more info!
